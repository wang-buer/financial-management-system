# 家庭理财管理系统



## 1、项目介绍

家庭理财管理系统拥有多种角色，可以自行设置权限和用户等，主要功能有：

收入管理、支付管理、资产管理、负债详情、统计报表、家庭成员管理、用户管理等

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：html

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
